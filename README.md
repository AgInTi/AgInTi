# AgInTi

Static company landing site for AgInTi, also known as `銀銦釱科技股份有限公司`.

Positioning: autonomous laboratory intelligence for chemistry, biomedicine, and scientific operations.

Custom domain: `https://agent.lazying.art/`

The GitHub Pages site lives in `docs/`:

- `docs/CNAME` configures the GitHub Pages custom domain.
- `docs/index.html` contains the landing page structure.
- `docs/styles.css` contains the visual system and responsive layout.
- `docs/app.js` powers the English / Traditional Chinese language dropdown.
- `docs/assets/aginti-logo.png` is the shared logo used by the header, hero card, and favicon.
- The landing page links to AgInTiFlow at `https://github.com/lazyingart/AgInTiFlow`.
