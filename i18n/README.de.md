# AgInTi Website

Helle Landingpage für **AgInTi** (`銀銦釱科技股份有限公司`) mit Fokus auf autonome Laborintelligenz für Chemie, Biomedizin und wissenschaftliche Abläufe.

Website: `https://agent.lazying.art/`

## Überblick

| Bereich | Richtung |
| --- | --- |
| Fokus | Autonome Labore, Chemie, Biomedizin |
| Produkt | AgInTiFlow auf GitHub |
| Deployment | GitHub Pages aus `docs/` |
| Domain | `docs/CNAME` -> `agent.lazying.art` |
| Frontend | HTML, CSS und JavaScript ohne Abhängigkeiten |

## Lokale Vorschau

```bash
cd /home/lachlan/ProjectsLFS/Agent/AgInTiLanding
python3 -m http.server 4174 --directory docs
```

Öffne `http://127.0.0.1:4174`.

## Visuelle Richtung

- Kurze Top-Botschaft: autonome Labore, schnellere Entdeckung.
- Lebendige Farben, saubere Abstände, moderne Lab-Tech-Ästhetik.
- AgInTiFlow wird als erstes öffentliches Produkt gezeigt.
