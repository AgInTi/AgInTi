# AgInTi Inc. Website

**AgInTi Inc.**(`銀銦釱科技股份有限公司`)의 밝은 랜딩 사이트입니다. 화학, 바이오메디컬, 과학 운영을 위한 자율 실험실 인텔리전스를 중심으로 합니다.

사이트: `https://agent.lazying.art/`

## 개요

| 영역 | 방향 |
| --- | --- |
| 초점 | 자율 실험실, 화학, 바이오메디컬 |
| 제품 | GitHub의 AgInTiFlow |
| 배포 | `docs/` 기반 GitHub Pages |
| 도메인 | `docs/CNAME` -> `agent.lazying.art` |
| 프론트엔드 | 의존성 없는 HTML, CSS, JavaScript |

## 로컬 미리보기

```bash
cd /home/lachlan/ProjectsLFS/Agent/AgInTiLanding
python3 -m http.server 4174 --directory docs
```

`http://127.0.0.1:4174` 를 여세요.

## 시각 방향

- 상단 메시지는 짧게: 자율 실험실, 더 빠른 발견.
- 밝고 생생한 색, 깨끗한 여백, 현대적인 랩테크 느낌.
- AgInTiFlow를 첫 공개 제품으로 소개합니다.
