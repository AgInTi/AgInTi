# AgInTi Inc. Website

AgInTi Inc.（`銀銦釱科技股份有限公司`）的明亮 landing site，定位为面向化学、生物医学与科研运营的自主实验室智能。

正式网站：`https://agent.lazying.art/`

## 快照

| 领域 | 方向 |
| --- | --- |
| 公司焦点 | 自主实验室、化学工作流、生物医学研究运营 |
| 产品链接 | GitHub 上的 AgInTiFlow |
| 部署 | GitHub Pages，来源为 `docs/` |
| 自定义域名 | `docs/CNAME` -> `agent.lazying.art` |
| 前端 | 无依赖 HTML、CSS、JavaScript |

## 本地预览

```bash
cd /home/lachlan/ProjectsLFS/Agent/AgInTiLanding
python3 -m http.server 4174 --directory docs
```

打开 `http://127.0.0.1:4174`。

## 内容方向

- 顶部文字保持简洁：自主实验室，更快发现。
- 使用明亮鲜明色彩、干净间距、现代实验室科技感。
- 提及 AgInTiFlow 作为第一个公开产品层。
