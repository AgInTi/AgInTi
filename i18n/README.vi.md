# AgInTi Website

Landing site sáng cho **AgInTi** (`銀銦釱科技股份有限公司`), tập trung vào trí tuệ phòng thí nghiệm tự động cho hóa học, y sinh và vận hành khoa học.

Trang web: `https://agent.lazying.art/`

## Tổng quan

| Khu vực | Hướng |
| --- | --- |
| Trọng tâm | Phòng thí nghiệm tự động, hóa học, y sinh |
| Sản phẩm | AgInTiFlow trên GitHub |
| Triển khai | GitHub Pages từ `docs/` |
| Tên miền | `docs/CNAME` -> `agent.lazying.art` |
| Frontend | HTML, CSS, JavaScript không phụ thuộc |

## Xem cục bộ

```bash
cd /home/lachlan/ProjectsLFS/Agent/AgInTiLanding
python3 -m http.server 4174 --directory docs
```

Mở `http://127.0.0.1:4174`.

## Hướng hình ảnh

- Thông điệp đầu trang ngắn: phòng thí nghiệm tự động, khám phá nhanh hơn.
- Màu sáng, bố cục sạch, phong cách lab-tech hiện đại.
- AgInTiFlow là sản phẩm công khai đầu tiên.
