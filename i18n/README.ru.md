# AgInTi Website

Яркий landing site для **AgInTi** (`銀銦釱科技股份有限公司`) с фокусом на автономный лабораторный интеллект для химии, биомедицины и научных операций.

Сайт: `https://agent.lazying.art/`

## Обзор

| Область | Направление |
| --- | --- |
| Фокус | Автономные лаборатории, химия, биомедицина |
| Продукт | AgInTiFlow на GitHub |
| Деплой | GitHub Pages из `docs/` |
| Домен | `docs/CNAME` -> `agent.lazying.art` |
| Frontend | HTML, CSS и JavaScript без зависимостей |

## Локальный просмотр

```bash
cd /home/lachlan/ProjectsLFS/Agent/AgInTiLanding
python3 -m http.server 4174 --directory docs
```

Откройте `http://127.0.0.1:4174`.

## Визуальное направление

- Короткий верхний текст: автономные лаборатории, более быстрые открытия.
- Яркие цвета, чистые интервалы, современная lab-tech эстетика.
- AgInTiFlow представлен как первый публичный продукт.
