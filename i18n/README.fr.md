# AgInTi Inc. Website

Site vitrine lumineux pour **AgInTi Inc.** (`銀銦釱科技股份有限公司`), positionné autour de l'intelligence de laboratoire autonome pour la chimie, la biomédecine et les opérations scientifiques.

Site: `https://agent.lazying.art/`

## Aperçu

| Domaine | Direction |
| --- | --- |
| Focus | Laboratoires autonomes, chimie, biomédecine |
| Produit | AgInTiFlow sur GitHub |
| Déploiement | GitHub Pages depuis `docs/` |
| Domaine | `docs/CNAME` -> `agent.lazying.art` |
| Frontend | HTML, CSS et JavaScript sans dépendances |

## Aperçu local

```bash
cd /home/lachlan/ProjectsLFS/Agent/AgInTiLanding
python3 -m http.server 4174 --directory docs
```

Ouvrez `http://127.0.0.1:4174`.
