# AgInTi Website

Landing site brillante para **AgInTi** (`銀銦釱科技股份有限公司`), centrado en inteligencia de laboratorio autónomo para química, biomedicina y operaciones científicas.

Sitio: `https://agent.lazying.art/`

## Resumen

| Área | Dirección |
| --- | --- |
| Enfoque | Laboratorios autónomos, química, biomedicina |
| Producto | AgInTiFlow en GitHub |
| Despliegue | GitHub Pages desde `docs/` |
| Dominio | `docs/CNAME` -> `agent.lazying.art` |
| Frontend | HTML, CSS y JavaScript sin dependencias |
