# AgInTi Website

موقع هبوط مشرق لـ **AgInTi** (`銀銦釱科技股份有限公司`) يركز على ذكاء المختبرات الذاتية للكيمياء والطب الحيوي والعمليات العلمية.

الموقع: `https://agent.lazying.art/`

## لمحة

| المجال | الاتجاه |
| --- | --- |
| التركيز | مختبرات ذاتية، كيمياء، طب حيوي |
| المنتج | AgInTiFlow على GitHub |
| النشر | GitHub Pages من `docs/` |
| النطاق | `docs/CNAME` -> `agent.lazying.art` |
| الواجهة | HTML و CSS و JavaScript بلا اعتماديات |

## معاينة محلية

```bash
cd /home/lachlan/ProjectsLFS/Agent/AgInTiLanding
python3 -m http.server 4174 --directory docs
```

افتح `http://127.0.0.1:4174`.

## الاتجاه البصري

- رسالة علوية قصيرة: مختبرات ذاتية، اكتشاف أسرع.
- ألوان مشرقة، مسافات نظيفة، وإحساس حديث بتقنية المختبرات.
- يظهر AgInTiFlow كأول منتج عام.
